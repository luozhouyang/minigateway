MiniGateway is built around a modular architecture that separates concerns cleanly.

## High-Level Architecture

The system consists of three main layers:

1. **Proxy Engine** - Handles incoming requests and routes them to backends
2. **Admin API** - RESTful interface for configuration management
3. **Web Dashboard** - React-based UI for visual administration
